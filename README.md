🚀 Club Connect Hub
Club Connect Hub is a modern, type-safe web application designed to streamline community engagement and club management. Built with a focus on performance and scalability, it leverages a cutting-edge tech stack to provide a seamless user experience for both administrators and members.
🛠 Tech Stack
This project demonstrates proficiency in modern full-stack development and cloud infrastructure:
• Frontend: React / TypeScript (96% of codebase)
• Routing & Logic: TanStack & Nitro
• Backend/Database: Supabase (PostgreSQL & Auth)
• Deployment: Cloudflare Workers (via Wrangler)
• Environment: Bun (Fast JavaScript runtime)
✨ Key Features
• Real-time Communication: Integrated chat features for instant member engagement.
• Type-Safe Architecture: Developed with TypeScript to ensure code reliability and minimize runtime errors.
• Serverless Deployment: Optimized for global performance using Cloudflare’s edge network.
• Secure Authentication: User management and data protection handled via Supabase Auth.

🚀 Installation & Local Development
To run this project locally, ensure you have Bun installed.

1. Clone the repository:
bash
git clone https://github.com/G-TechFred/club-connect-hub.git
cd club-connect-hub

3. ⁠2. Install dependencies:
bash
bun install

3. Set up Environment Variables:
Create a .env file based on
the .env.example provided and add your Supabase/Cloudflare credentials.

4. Run the development server:
bash
bun dev
